# üéØ Quick Installation Checklist

## For Each New Restaurant Client

### Phase 1: WordPress Setup (10 minutes)
- [ ] Install WordPress on hosting
- [ ] Set up database
- [ ] Install and activate "Uncle Chan's Chicken" theme
- [ ] Delete default posts/pages
- [ ] Set permalink structure to "Post name"

### Phase 2: Branding (15 minutes)
- [ ] Go to **Restaurant ‚Üí General Settings**
- [ ] Enter restaurant name
- [ ] Add tagline/slogan
- [ ] Set primary color (their brand color)
- [ ] Set accent color
- [ ] Add phone number
- [ ] Add email address
- [ ] Upload logo (Appearance ‚Üí Customize)

### Phase 3: Menu Setup (30-60 minutes)
- [ ] Go to **Menu Items ‚Üí Categories**
- [ ] Create categories: Chicken, Sides, Drinks, Desserts, etc.
- [ ] Go to **Menu Items ‚Üí Add New**
- [ ] For each item:
  - Add title
  - Upload image (high quality!)
  - Write description
  - Set price
  - Select category
  - Add calories (optional)
  - Set spice level (if applicable)
  - Add sides options (name + price)
  - Add extras options (name + price)
  - Mark as Popular/New/Featured
  - Publish

### Phase 4: Location & Hours (10 minutes)
- [ ] Go to **Locations ‚Üí Add New**
- [ ] Enter location name
- [ ] Add full address
- [ ] Enter phone number
- [ ] Add email
- [ ] Set business hours
- [ ] Add delivery radius (in miles)
- [ ] Get lat/long from Google Maps (optional)
- [ ] Publish

### Phase 5: Special Offers (15 minutes)
- [ ] Go to **Special Offers ‚Üí Add New**
- [ ] Create 2-3 offers:
  - Welcome offer (10% off first order)
  - Family deal (combo special)
  - Limited time promotion
- [ ] For each offer:
  - Add title (e.g., "New Customer Special")
  - Write description
  - Upload enticing image
  - Set discount type (percentage or fixed)
  - Set discount amount
  - Create promo code (e.g., WELCOME10)
  - Set expiry date
  - Check "Featured" for best offer
  - Publish

### Phase 6: Customer Reviews (20 minutes)
- [ ] Go to **Reviews ‚Üí Add New**
- [ ] Add 5-10 testimonials
- [ ] For each review:
  - Customer name
  - Review text (positive, specific)
  - 5-star rating
  - Customer location (city, state)
  - What they ordered
  - Check "Verified Purchase"
  - Upload customer photo (or use stock)
  - Publish

### Phase 7: Social Media (10 minutes)
- [ ] Go to **Restaurant ‚Üí Social Media**
- [ ] Add social media URLs:
  - Facebook page
  - Instagram profile
  - Twitter/X handle
  - YouTube channel
  - TikTok account
- [ ] Get Instagram access token (optional):
  - Go to Facebook Developers
  - Create app ‚Üí Instagram Basic Display
  - Get access token
  - Paste in settings
  - Set number of posts to show (6-12)
  - Add featured hashtag
- [ ] Save settings

### Phase 8: Email Popup (10 minutes)
- [ ] Go to **Restaurant ‚Üí Email Popup**
- [ ] Check "Enable Popup"
- [ ] Set delay (5-10 seconds recommended)
- [ ] Customize title: "[Restaurant] Family Club"
- [ ] Write description
- [ ] Set discount badge text (10% OFF)
- [ ] Button text: "Join Now" or "Claim My Discount"
- [ ] Enable exit-intent (optional)
- [ ] Save settings

### Phase 9: Delivery Settings (5 minutes)
- [ ] Go to **Restaurant ‚Üí Delivery**
- [ ] Check "Enable Delivery"
- [ ] Set delivery fee ($3-$6)
- [ ] Set minimum order ($15-$25)
- [ ] Set free delivery threshold ($30-$50)
- [ ] Estimated delivery time (30-45 minutes)
- [ ] Add Google Maps API key (optional, for maps)
- [ ] Save settings

### Phase 10: Final Touches (15 minutes)
- [ ] Test menu ordering flow
- [ ] Check mobile responsiveness
- [ ] Test email popup
- [ ] Verify all links work
- [ ] Check Instagram feed loads
- [ ] Test special offer code copy
- [ ] Review all spelling/grammar
- [ ] Take screenshots for portfolio
- [ ] Get client approval

### Phase 11: Training Client (20 minutes)
Show them how to:
- [ ] Add new menu items
- [ ] Update prices
- [ ] Create special offers
- [ ] Moderate reviews
- [ ] View email subscribers
- [ ] Update business hours
- [ ] Change photos

## üö® Common Issues & Fixes

### Images not uploading
- Increase PHP upload limit in hosting
- Resize images to under 2MB
- Use JPG format

### Instagram feed not showing
- Verify access token is valid
- Check token hasn't expired (refresh every 60 days)
- Confirm Instagram account is public

### Email popup not appearing
- Clear browser cache
- Check "Enable Popup" is checked
- Verify delay setting isn't too long
- Check cookie hasn't been set (incognito mode)

### Menu items not displaying
- Verify item is Published (not Draft)
- Check category is assigned
- Confirm image is uploaded
- Clear cache
